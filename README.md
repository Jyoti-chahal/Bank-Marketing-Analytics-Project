# Bank Marketing Analytics – End-to-End Project

This repository contains a complete end-to-end data analytics project using Python, SQL (MSSQL), and Power BI.

## Project Objective
To analyze bank marketing campaign data, understand customer behavior, and evaluate factors influencing term deposit subscriptions.

## Tools & Technologies
- Python (pandas, numpy, matplotlib, seaborn, scikit-learn)
- Microsoft SQL Server (T-SQL)
- Power BI Desktop & Power BI Service
- GitHub

## Project Structure
- data/ → raw and cleaned datasets  
- python/ → EDA, statistics, ML models  
- sql/ → queries, views, indexes, procedures  
- powerbi/ → dashboards and Power BI Service analysis  

## Python Analysis
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Statistical analysis (Welch t-test, Pearson correlation)
- Machine learning models (Logistic Regression, Random Forest)
- Model evaluation (accuracy, precision, recall, ROC-AUC)

## SQL Analysis
- Complex queries (Top jobs by balance, campaign success rate)
- Views for client summary and campaign performance
- Index creation for performance optimization
- Stored procedure for balance updates

## Power BI Analysis
- Interactive dashboards
- DAX measures and calculated columns
- Demographic, loan, and campaign performance analysis

## Power BI Service & Security
- Report published to Power BI Service
- Dashboard created from pinned visuals
- Scheduled refresh configured
- Row-Level Security (RLS) implemented
- Data alerts configured on KPI cards

## Key Insights
- Campaign success varies by month and customer demographics
- Housing loan customers show higher subscription rates
- Excessive campaign contacts reduce success probability
